# Card Haven

A modern, feature-rich PWA webstore built with Python and Flask.
Includes PWA support, offline caching, and an admin/seller interface and fun easter eggs
## Author

- [Benjamin Robertson (Developer)](https://github.com/Robertson-B)



## Features

Backend: Flask + sqlite3 (small portions optionally use SQLAlchemy via `models.py`)

Frontend: Jinja2 templates + Bootstrap 5, plus a small amount of custom CSS/JS

PWA: `static/manifest.json` and `static/sw.js` provide offline fallback and caching

Testing accounts provides. username and password fong for user account. username and password admin for test admin and seller account.
## Installation
Clone the repository

Pip install the requirements.txt file

Run the app.py file to run the server
